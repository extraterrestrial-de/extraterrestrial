\doxysubsection{Detailed Description}
A polygon in a 2D grid

The area is defined by a collection of points (also known as polygons). The points have to be stored in a specific order to prevent mistakes. The edge a is defined by line(p\+\_\+s\mbox{[}i\mbox{]}, p\+\_\+s\mbox{[}i+1\mbox{]}) with the only exception being the last element of p that is connected with the first one
