\doxysubsection{Detailed Description}
A type of \mbox{\hyperlink{class_p_vector}{PVector}} that can be collected by a robot.

Collectibles have a color value in addition to normal points in 2D-\/plane. They can be picked up by the robot in the Co\+Space-\/\+SE. However the robot can\textquotesingle{}t interact directly with this program and the map the robot uses is actually created by the {\bfseries{map\+\_\+interpreter.\+py}} script. This can result in imprecise movement and calculations. The method {\bfseries{\mbox{\hyperlink{class_collectible_a068a4938bf6bb68cf6e7b1fd33dd1f4c}{Collectible\+::is\+Correct\+Collectible()}}}} resolves this problem taking the imprecision into account.
